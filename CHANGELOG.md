# 发行日志 (Changelog)

本文档记录「GUI 刷题与考试程序」的所有版本变更与开发记录。

---

## [v2.5.0] - 2026-06-18

### ✨ 新增：外部题库路径支持
- **题库文件独立存储**：所有新建题库默认存入 `banks/` 文件夹
- **添加外部题库文件**：题库管理窗口新增「📄 添加题库文件」→ 选择任意位置的 `.json` 题库文件
- **扫描文件夹导入**：新增「📂 扫描文件夹」→ 自动识别目录内所有有效题库 JSON 并批量添加
- **智能路径管理**：
  - 注册表记录每个题库的实际文件路径（不再是名称派生）
  - 外部题库文件不会被删除/重命名物理文件（仅管理 BANKS_DIR 内的文件）
  - 支持同名文件从不同路径添加（自动重命名）

## [v2.4.0] - 2026-06-18

### ✨ 新增
- **按题型设置考试题量**：考试设置窗口改为分别设置单选题/多选题/填空题数量，显示各题型可用题数
- **错题本功能**：
  - 做错的题目自动记入错题本（`banks/<name>_wrong.json`），累计错误次数
  - 刷题入口弹出选择：正常刷题 / 错题本刷题（显示错题数）
  - 练习和考试模式均自动记录错题
  - 刷题界面显示当前题目的历史错误次数
- **键盘快捷键**：
  - ← → 左右箭头切换题目（练习和考试模式均支持）
  - A/B/C/D 字母键直接选择对应选项（单选切换、多选勾选/取消）
  - 回车键提交答案（练习模式）
  - 顶部提示栏显示快捷键说明

### 🔧 调整
- **按钮文字精简**：去掉所有括号注释（"刷题模式（练习）"→"刷题模式"等）
- **字体加大**：
  - 大布局主菜单按钮：11pt → 12pt（小四）
  - 练习/考试题目文本：12pt → 14pt（四号）
  - 选项和标签同步增大

## [v2.3.0] - 2026-06-18

### ✨ 新增：多题库管理
- **题库管理器 `BankManager`**：支持创建、切换、删除、重命名多个独立题库
  - 题库存储在 `banks/` 目录，每个题库独立的 JSON + 历史文件
  - 注册表 `banks/index.json` 记录所有题库及当前活跃题库
  - 首次运行自动创建「默认题库」
- **主窗口题库切换**：
  - 标题栏显示当前题库名称和统计信息
  - 「🔄 切换题库」按钮 → 弹出题库管理窗口
  - 管理窗口支持：切换、创建、重命名、删除题库
- **导入目标选择**：
  - 导入窗口新增「导入目标」选项组
  - 可选：添加到当前题库 / 创建新题库 / 添加到已有题库
  - 创建新题库时自动生成名称（导入题库_月日_时分）

## [v2.2.2] - 2026-06-18

### 🐛 修复
- **空格导致解析失败**：`parse_txt` 全部正则改为高度容错模式
  - 题型行：`1 . ( 单选题 ) 题目` 也能识别（`\s*` 填充各处）

  - 填空行：`空 1 : 答案` 带空格也能解析
  - 不再因任何空白字符导致导入失败

## [v2.1.2] - 2026-06-18

### 🐛 修复
- **大型布局按钮间距不正确**：两个问题导致间距比例失调
  1. `pady` 使用固定值（8px），未随窗口缩放重新计算 → 改为从可用垂直空间动态推导：`btn_h = avail_h / (n_btns + n_gaps*0.5)`，`pady = btn_h/4`
  2. `_update_layout` 仅在模式切换时调用 apply → 改为大型模式下每次 resize 都重新计算间距
- 紧凑模式不受影响（固定尺寸，仅在模式切换时应用一次）

## [v2.1.1] - 2026-06-18

- **主窗口按钮间距精细化**：

## [v2.0.0] - 2026-06-18

  - 支持提前提交试卷
- **成绩历史记录**：

  - 统计窗口显示历史考试列表（日期、题量、正确数、正确率、用时）
  - 计算历史平均正确率
- **题库管理 GUI**：
  - Treeview 列表展示所有题目（序号、题型、预览）
  - 实时搜索过滤
  - 双击编辑题目

  - 保留 `Question`(ABC) / `SingleChoiceQuestion` / `MultiChoiceQuestion` / `FillInBlankQuestion` 数据模型
  - `QuestionBank` 类增加 `history` 历史记录支持

## [v1.0.0] - 2026-06-18

### 🎉 首次发布

#### 新增功能
- **三种题型支持**：单选题、多选题、填空题（含多空、多答案匹配）
- **题库管理**：支持题目的显示、添加、删除、修改、关键词搜索
- **JSON 持久化**：题库以 JSON 格式自动保存/加载（`questions.json`）
- **TXT 导入**：支持从 TXT 文件导入或直接粘贴文本，按规范格式解析
- **TXT 导出**：将当前题库按规范格式导出为 TXT 文件
- **交互式刷题**：
  - 随机打乱题目顺序，每次显示一道
  - 支持键盘 ← → 切换题目（需 `keyboard` 库）
  - 无 `keyboard` 库时自动降级为输入指令模式（n/p 切换）
  - H 键显示/隐藏正确答案
  - Q 键退出刷题并查看统计报告
- **统计功能**：题库总览统计 + 刷题成绩报告（正确率、各题型正确率）
- **依赖检查**：启动时检测 `keyboard` 库可用性，缺失时给出安装提示

#### 技术架构
- 面向对象设计：`Question`(ABC) / `SingleChoiceQuestion` / `MultiChoiceQuestion` / `FillInBlankQuestion`
- 题库管理：`QuestionBank` 类
- 刷题引擎：`ExamEngine` 类
- 纯 Python 标准库 + 可选 `keyboard` 第三方库

#### 文件清单
| 文件 | 说明 |
|------|------|
| `main.py` | 主程序（GUI 版，约 700 行） |
| `requirements.txt` | Python 依赖声明（零依赖，仅标准库） |
| `CHANGELOG.md` | 发行日志（本文件） |
| `questions.json` | 题库数据（自动生成） |
| `history.json` | 考试历史记录（自动生成） |
